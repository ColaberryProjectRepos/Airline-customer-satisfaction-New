Methodology 3
Data Preparation

The data preparation process is crucial for ensuring high-quality input for machine learning models, improving prediction accuracy, and enhancing the overall reliability of insights.


Model Selection :

To accurately predict airline customer satisfaction, multiple machine learning algorithms were tested, including ExtraTreesClassifier, Logistic Regression, Random Forest, and XGBoost. Since this is a binary classification problem (Satisfied vs. Dissatisfied), the models were evaluated using key performance metrics such as accuracy, precision, recall, and F1-score. The ExtraTreesClassifier was chosen because it effectively identifies the most influential factors affecting customer satisfaction, enabling airlines to make data-driven improvements in inflight services, customer experience, and operational strategies.


Model Development:

The airline customer satisfaction prediction model was developed using a structured machine learning pipeline, ensuring accurate and reliable predictions. The ExtraTreesClassifier, which achieved 98.9% accuracy, was selected as the optimal model after testing multiple algorithms, including Logistic Regression, Random Forest, and XGBoost. Feature importance analysis was conducted, revealing that seat comfort, inflight entertainment, Wi-Fi quality, and Food and Drink were among the top contributors to customer dissatisfaction.
