 Adding Canvas Background

     1. From the report view, go to Visualization pane -> Format-> Canvas  Background-> Browse the             image
     2. Choose Transparency as 30%
     3. Change Image Fit as 'Fit'



Adding colaberry logo

    1   First , saved the logo on my devise 
    2   Access the "Insert" tab on the Power BI ribbon and select the "Image" icon to insert an image               onto your report canvas.
    3   Choose the Logo Image
    4    In the Format Pane on the right, Go to General => Properties and enter            the below                   values for formatting

            General=>   Height, Width = 165,Horizontal = 1,Vertical = 3


Adding Title

1.Click on INSERT from the ribbon.
2. In the Format Pane on the right, Go to General => Properties and enter            the below values          for formatting
       General=>
             Height = 66, Width = 551, Horizontal = 377, Vertical = 1
    Title => On => Type " JET AIRWAYS CUSTOMER SATISFACTION ", Size 20, Color= #252423
