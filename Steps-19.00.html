To create Dissatisfied by Service Type and Rating visual in Power BI using the 100% Stacked Column Chart.
  Follow these steps:
   From the Visualizations pane, select the '100% Stacked Column Chart' and place it in the designated area of your Power BI report.
  1. Assign 'Service Type' to the X-axis to categorize data. For the column values on the Y-axis, use 'Sum of Total Customers' 
  2. Add "Rating Bucket" on the Legend on each score  
  3. If (score <= 3,"1. Low Rating", else"2. High Rating")
  4. Name your chart 'Distribution of Dissatisfied by Service type and Rating' and adjust the visual's title accordingly.
      Fill up figures provided below for the necessary fields.
      General: Properties, Size,
       Height=234, Width=632, Position, Horizontal=47, Vertical=222Title=Off, column
        Apply setting to "High Rating" = #789E11  "green"
                                   "Low Rating" = #D64550 "red"
       Effects, Background =On, Color=#E6E6E6,Transparency=0%, Visual border=off
      Visual: Callout Value, Font=Segeo UI (10), Bold, Font color= #605E5C (black), Category label=Off
      Display Units-Auto, Value decimal places-Auto
      Category Label-Off
