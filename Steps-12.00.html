Create DAX Measures 12

Steps :

  1. Go to “Report view” in Power BI.
  2. Right-click on the "Airline customer Satisfaction" table in the Fields pane.
  3. Select "New Measure" to create a new measure.
  4. Use the provided DAX formulas to create the Measures, KPI's, Sub-KPI's and Arrow and save them.
  5. Repeat the above steps for each of the desired Measures, KPI's, Sub-Kips and Arrows.
